HapYak API Exploration/Demo

This page demonstrates various advanced HapYak features, including variables and interaction with the API. All of the JavaScript is contained in the file custom.js

This demo uses jQuery for easier DOM interaction and materialize for some css components and JS functionality.

Created by bobaks@google.com | HapYak Project link

Example 1: Set a variable in the HapYak viewer

In this example, the text entered below is passed to the HapYak viewer and stored in the variable name. This value can then be used in an annotation or a display rule. In this example, once the variable is set, it is used in an annotation. Several annotations have their display rules to be set only if this value exists (otherwise, the annotations are hidden).

Example 4: Messages

This uses the hapyak.message.send() to send data from an iframe annotation to the parent page.

Example 5: Addon

This example uses the hapyak.addon() method. The addon is defined in the JS, and then called from an annotation. This functions similarly to messages, but doesn't require an iframe annotation and can be added to any annotation.

In this example, the time trigger annotation triggers the addon. (It is set to change the text at 5s, 10s, and 15s)
